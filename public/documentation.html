Action Request URL Response
Return all movies GET '/movies' Returns an array of all JSON movie objects in the database
Return single movie data GET '/movies/:title' Returns a JSON object returning the data for one movie based on the title passed in the url parameter
Return data about a genre GET '/movies/:genre' Returns an array of all JSON movie objects within database that includes the genre passed in the url parameter
Return data about a director GET '/movies/:directorName' Returns a JSON object containing data of the director based on the name that was passed into the url parameter
Allow new users to register POST '/users' Returns the same JSON object that was passed into the req body and added to the database
Allow users to update their user info PUT '/users/:username' Returns the data fields that was updated by the user
Allow users to add a movie to their list of favorites POST '/users/add/:movieName' Returns a text confirming which movie was added successfully
Allow users to remove a movie to their list of favorites DELETE '/users/remove/:movieName' Returns a text confirming which movie was removed successfully
Allow existing users to deregister DELETE '/users/deleteAccount' Returns a text confirming that the user's email was deleted successfully
